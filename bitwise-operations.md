# Bitwise Operations


## Bitwise Operators
| byte-1   | operation    | byte-2   | result |
| -------- | ------------ |--------- | ------ |
| `1010`   | `&`  and     | `0011`   | `0010` |
| `1010`   | `\|`  or     | `0011`   | `1011` |
| `1010`   | `^`  xor     | `0011`   | `1001` |
| `1010`   | `&^` and not | `0011`   | `1000` |
|          | `^`  not     | `1010`   | `0101` |

## Bitshift Operators
| number-1   | operation  | number-2 | result |
| ---------- | ---------- | -------- | ------ |
| `1000`     | `>>` right | 2        | `0010` |
| `0001`     | `<<` left  | 2        | `0100` |
| 1          | `<<`       | 2        | 4      |
| 1          | `<<`       | 4        | 16     |
