** Overview

- UTF-8 :: Unicode Transformation Format. 8-bit base size.

- Encodes up to 4 bytes

- ASCII compatible

- Self synchronizing

- Compressible

#+begin_example
  === Bit Base Sizes ===

           +---+---+---+---+
  UTF-8  = | x |   |   |   |
           +---+---+---+---+
           +---+---+---+---+
  UTF-16 = | x | x |   |   |
           +---+---+---+---+
           +---+---+---+---+
  UTF-32 = | x | x | x | x |
           +---+---+---+---+
#+end_example
