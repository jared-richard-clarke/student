# Two's Complement

Source: [Two's Complement](https://en.wikipedia.org/wiki/Two%27s_complement)

## Three-Bit Integers

| Bits  | Unsigned | Signed |
| ----- | -------- | ------ |
| `000` | 0        | 0      |
| `001` | 1        | 1      |
| `010` | 2        | 2      |
| `011` | 3        | 3      |
| `100` | 4        | -4     |
| `101` | 5        | -3     |
| `110` | 6        | -2     |
| `111` | 7        | -1     |
